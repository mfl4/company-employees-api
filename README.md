## Karyawan API Spec

### Create Karyawan API

Endpoint : GET /api/karyawan

Request Body :

```json
{
  "nama": "Mhd. Farhan Lubis",
  "alamat": "Jl. Siliwangi"
}
```

Response Body Success :

```json
{
  "nama": "Mhd. Farhan Lubis",
  "alamat": "Jl. Siliwangi"
}
```

Response Body Erorr :

```json
{
  "error": "name is required"
}
```
